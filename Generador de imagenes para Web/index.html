Generador de imagenes

Servicio de generación de imágenes para las publicaciones
pasos a seguir:
1- Presionar el botón seleccionar archivo
2- Seleccionar el archivo de su computadora que desea añadir a la publicación
3- Presionar el botón generar Link
Demorará unos instantes en generar el link, una vez creado se mostrará el link generado.
4- Presionar el botón copiar al portapapeles para copiar el link generado.
5- Agregar el link en la publicación utilizando el botón derecho del del mouse y seleccionado Pegar
